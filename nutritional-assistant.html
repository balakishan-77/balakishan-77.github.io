Neutrino 

Managing nutrition for better health

Neutrino is a RAG based Agentic AI system which provides nutrition assistance for managing nutritional disorders. It integrates literature in nutrition with Gen AI technologies to provide swift, accurate, and contextually relevant responses to queries on nutrition and its related disorders. 

The system takes historical interactions into account while asking for clarifications when needed.

Chunking 

LamaParse is used to generate  semantic chunks as part of chunking strategy.

Agentic clarity

The react agent uses groundedness and precision metrics for validation of generated reponse.

HyPE

The system generates HyPE questions for each chunk.  

Retrieval

Given a user query, the system finds top 5 HyPE questions which semantically match the user query.

Generation

Respective chunks are retrieved corresponding to the matched HyPE questions.

How to use Neutrino

  • 1. Share the health background—diseases, lab values, medications, and dietary preferences.
  • 2. Pose a goal or question—be it therapeutic, preventive, or educational.
  • 3. Review the generated response.
