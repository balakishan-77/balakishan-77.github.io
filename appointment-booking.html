RASA Appointment Bot

Conversational scheduling powered by RASAPRO

RASA appointment bot is a booking assistant which orchestrates the full dialog flow—from authentication to doctor selection to appointment booking— while using LLM reasoning to keep the experience natural to patients.

Powered by RASAPRO, it understands context, refers to past interactions, and validates every step before confirming a slot.

Secure OTP

Patients authenticate through a trusted OTP channel

Context-Aware Search

Supports filtering by specialty leveraging integrated RAG

Streamlined Booking

Users confirm their slot through intuitive prompts designed as part of dialog flow

Dialog Transparency

Each suggested action carries a rationale and link to previous exchanges.

Integration

Integrates with external systems during the dialog flow

Software Architecture

Deployment Architecture

How to use the assistant

  • 1. Share the care need—speciality, date range, or preferred doctor.
  • 2. Authenticate via the OTP so we can retrieve personal calendars.
  • 3. Confirm the slot; the assistant sends confirmations and reminders automatically.

The assistant supports multiple conversational channels and can be integrated into existing systems.
